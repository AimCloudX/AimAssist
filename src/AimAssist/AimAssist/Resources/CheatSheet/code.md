# ファイル操作
- 新規: Ctrl + N 
- 開く: Ctrl + O 
- 保存: Ctrl + S 
- 名前を付けて保存: Ctrl + Shift + S 
- ファイルを閉じる: Ctrl + W 

# 基本編集
- カット: Ctrl + X 
- コピー: Ctrl + C 
- ペースト: Ctrl + V 
- 元に戻す: Ctrl + Z 
- やり直し: Ctrl + Y 
- すべて選択: Ctrl + A 

# ナビゲーション
- コマンドパレット: Ctrl + Shift + P 
- クイックアクセス: Ctrl + P 
- 最近開いたファイル: Ctrl + R 
- 行に移動: Ctrl + G 

# ビューとレイアウト
- エクスプローラーを表示: Ctrl + Shift + E 
- 検索を表示: Ctrl + Shift + F 
- Gitを表示: Ctrl + Shift + G 
- デバッグを表示: Ctrl + Shift + D 
- 拡張機能を表示: Ctrl + Shift + X 
- サイドバーの切り替え: Ctrl + B 
- ターミナルの表示: Ctrl +` (バッククォート)

# 編集
- 行の複製: Shift + Alt + ↓ または ↑ 
- 行の削除: Ctrl + Shift + K 
- コメントの切り替え: Ctrl + / 
- 行の移動: Alt + ↓ または ↑ 
- インデント: Ctrl + ] 
- アウトデント: Ctrl + [ 

# マルチカーソルと選択
- カーソル追加 (上/下): Ctrl + Alt + ↑ または ↓ 
- 次を選択: Ctrl + D 
- 前を選択: Ctrl + K Ctrl + D 
- 一致するすべてを選択: Ctrl + Shift + L 
- 行末にカーソル追加: Shift + Alt + I 

# 検索と置換
- 検索: Ctrl + F 
- 置換: Ctrl + H 
- 次を検索: F3 
- 前を検索: Shift + F3 
- 選択範囲で検索: Ctrl + F3 
- 次を置換: Ctrl + Shift + 1
