# ウィンドウ操作
- AimAssistの表示/非表示  Alt + A
- ピッカーウィンドウの表示  Alt + P
- AimAssistを閉じる  Ctrl + D

# ナビゲーション
- 次のユニット  Ctrl + N
- 前のユニット  Ctrl + P
- 次のモード  Ctrl + Shift + N
- 前のモード  Ctrl + Shift + P
- コンテンツにフォーカス  Ctrl + K, Ctrl + L
- ユニットにフォーカス  Ctrl + K, Ctrl + J

# ツール
- ショートカット設定を開く  Ctrl + K, Ctrl + S
